# homebrew-tap

This is a Homebrew tap containing formulae for my open-source projects.

## Available Formulae

### [Golazo](https://github.com/0xjuanma/golazo)

A minimal TUI for real-time football stats and scores.

```bash
brew install 0xjuanma/tap/golazo
```

### [Anvil](https://github.com/0xjuanma/anvil)

CLI to streamline config management and tool installation. Install your full dev tool-chain in one command.

```bash
brew install 0xjuanma/tap/anvil
```

### [Helm](https://github.com/0xjuanma/helm)

Minimalistic & customizable Pomodoro-like timer for pure focus.

```bash
brew install 0xjuanma/tap/helm
```
